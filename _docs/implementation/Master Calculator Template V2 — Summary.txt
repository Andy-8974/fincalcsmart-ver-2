9. What Must Not Be Changed Accidentally
math.ts — semi-annual compounding formula (monthlyRateCA), calcPayment, buildSchedule, getCmhcRate are the financial source of truth
id="expert-analysis" — scroll target used by "See How to Save" / "Unlock AI Analysis" buttons
fmtCAD / fmtCADx — all monetary display must go through these formatters
Health Score algorithm — GDS/TDS/DP scoring bands are calibrated, do not simplify
IIFE pattern in Block C — introduces computed vars (hScore, GR, GARC, GFIL, rStatus) inside the conditional render; removing it breaks variable scope
md:hidden / hidden md:flex split in Smart Opt card — mobile and desktop layouts are separate blocks; editing one does not affect the other
useRegion() hook — region state (us / ca) is global; never hardcode region logic
